# ROADMAP

Ce dépôt contient les schémas du flux de conversion.

## Fichiers
- `schemas-complets.html` : version HTML complète.
- `schema-mermaid.md` : version mermaid.
- `schema-visuel.svg` : export SVG.
- `conversion-flow-schema.jsx` : composant JSX du schéma.

## Prévisualisation via GitHub Pages
Une copie du HTML est dans `docs/index.html` pour GitHub Pages. Après activation de Pages (source : `main` / dossier `docs`), la page sera disponible à l’URL :
```
https://yohand-byte.github.io/ROADMAP/
```
Si Pages n’est pas encore activé, ouvre simplement le fichier brut :
```
https://raw.githubusercontent.com/yohand-byte/ROADMAP/main/schemas-complets.html
```
